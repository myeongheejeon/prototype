# 시안 A (Design A)

- **ver01** = 시안 C
- **design-a** = 시안 A

## 구성

- **배경**: 전체 화면 동영상 (그라데이션 오버레이)
- **중앙**: 큰 검색창 (플레이스홀더 로테이션, 검색 시 대화에 추가)
- **검색창 아래**: 바로 대화 가능한 칩 (클릭 시 검색창에 반영)
- **하단**: 태깅해서 대화할 수 있는 영역 (대화 목록 + 입력창)

## 실행

```bash
cd design-a
npm install
npm run dev
```

브라우저에서 http://localhost:3000 접속.

## 배경 동영상

배경용 동영상은 `ver01/public/화면 기록 2026-02-05 오후 3.24.08.mov` 를 `design-a/public/` 로 복사해 두면 됩니다. (이미 복사되어 있을 수 있음)
